This ToDo app is designed to help you organize and manage your tasks efficiently. You can add new tasks, update existing ones, mark tasks as completed, and delete tasks you no longer need.

How to Use the App

  • Add a New Task: On the home page, fill in the "Todo Title" and "Todo Description" fields, then click the "Add Todo" button. Your task will be saved and displayed in the task list.
  • Mark a Task as Completed: In the task list, click the "Complete" button next to a task to mark it as done. Completed tasks are highlighted in green and labeled as "Completed." Click "Undo" to mark it as incomplete again.
  • Update a Task: Click the "Update" button next to a task to edit its title or description. Save your changes by clicking "Update Todo."
  • Delete a Task: Click the "Delete" button next to a task to permanently remove it. Youâ€™ll be prompted to confirm the deletion.
  • Search Tasks: Use the search bar on the home page to find tasks by title or description.
  • Filter Tasks: Use the filter dropdown to view all, completed, or incomplete tasks.

Features

  • Modern and intuitive interface for seamless task management.
  • Mark tasks as completed with a single click, with visual feedback.
  • Persistent task storage in a database.
  • Quick actions to add, update, complete, and delete tasks.
  • Search and filter functionality to easily find tasks.
  • Responsive design for use on desktop and mobile devices.
  • Accessible interface with ARIA attributes for screen readers.
